# Tolmač

Forensic encoding decoder and verifier.

Auto-detects UTF-8, UTF-16 LE/BE, UTF-32, ASCII, BCD nibble-swap, Base64
and produces court-defensible step-by-step decode logs suitable for
evidentiary documentation under s.146 of the Evidence Act 1995 (Cth).

Part of the [Strategos Suite](https://strategos.au) by ShadowStrike.
